# Solar Powered Storage System for Perishable Foodstuffs in Nigeria

## 📌 Project Overview

Post-harvest losses in Nigeria account for 30–50% of perishable food production due to inadequate storage and unreliable electricity supply. 

This project proposes the design and implementation of a **solar-powered cold storage system** to preserve perishable foodstuffs such as tomatoes, vegetables, fruits, fish, and dairy products.

The system is designed to:
- Operate independently of grid power
- Reduce food spoilage
- Support rural farmers and markets
- Improve food security and income stability

---

## 🎯 Objectives

- Design a cost-effective solar-powered cold storage unit
- Ensure temperature control between 0°C – 10°C
- Provide off-grid energy independence
- Reduce post-harvest losses
- Develop scalable storage solutions for rural communities

### 1. Solar Power System
- Solar Panels
- Charge Controller
- Battery Bank
- Inverter

### 2. Cooling System
- DC Compressor
- Evaporator & Condenser
- Insulated Storage Chamber
- Temperature Monitoring System

### 3. Monitoring & Control
- Digital Thermostat
- IoT Monitoring (Optional)
- Battery Management System

---

## ⚙️ System Design

### Power Flow:
Solar Panels → Charge Controller → Batteries → Inverter → Cooling Unit

### Storage Capacity (Example Model)
- 500kg – 1 Ton Storage
- Temperature Range: 2°C – 8°C
- 24-hour Operation with Battery Backup
